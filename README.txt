Los requisitos necesarios para correr mi proyecto son:
-Visual Studio o Visual Studio Code.
-Net6.

Funcionalidades basicas:
-Busqueda de texto. En caso de no encontrarse con lo ingresado por el usuario se buscan frases o palabras parecidas.
-Implementados los operadores de cercania(~), negacion(!), siempre debe aparecer(^) y prioridad(*).
